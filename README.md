#  Classification des fleurs d'Iris
Contexte ce projet s'inscrit dans le cadre du module logiciels libres et open source (LLOS), niveau master 1 (microbiologie appliquee, biochimie appliquee, parasitologie, biotechnologie et valorisation des plantes).
il illustre la mise en place d'un workflow de recherche reproductible en utilisant Git/GitHub, Jupyter Notebook et des bibliotheques Python pour l'analyse de donnees.
